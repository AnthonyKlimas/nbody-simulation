# N-Body Simulation (C++)

Physics-based simulation modeling gravitational interactions between celestial bodies using C++ and object-oriented design.

---

## Features
- Simulates motion using Newtonian physics
- Object-oriented design with classes and inheritance
- Reads simulation data from input files
- Real-time visualization using SFML

---

## Technologies
- C++
- SFML
- Linux
- STL

---

## Concepts
- Object-Oriented Programming (OOP)
- Data Structures
- Numerical Methods
- File I/O Handling
- Systems Programming

---

## How to Run

```bash
make
./NBody <total_time> <time_step> <input_file>
